everything about results

/results Will return the results from today and yesterday.
/results/teams/[teamids] Will return the results from the past 3 months from teams in an array.
/results/events/[eventids] Will return the results from the past 3 months from events in an array.

everything about players

/player/name This will return a specified player for a given name.
/playerById/id Will do the same as above but takes an id instead of a name.
/playerstats/id This will give you additional information about a player./a>
/playerranking/ This gives current playerranking from this year(may take some time to load).
/playerranking/year This gives playerranking from specified year (may take some time to load).

everything about events

/events Will return all upcoming and ongoing events.
/event/name This gives info about a sepcific event.
/eventbyid/id This gives info about a sepcific event.
/pastevents Will return all past events from the last month.
/pastevents/team/[teamids] Will return all played events during the last year from an array of teams.
/pastevents/player/[teamids] Will return all played events during the last year from an array of players.

everything about teams

/team/name This gives achievments of a team.
/teambyid/id This gives achievments of a team.
/teamstats/id/ This gives stats about a team.

everything about teamranking

/ranking This Will return the current top30 world ranking.
/ranking/country Will return the current ranking in the specified country.
/ranking/day(number)/month(word)/year(number)Will return the world ranking at the specified date.

everything about matches

/matches This data contains information about all currently scheduled matches.
/match/id This gives infomation about a match.
/matchstats/id This will give additional information about a match.
/matchesstats Will return information about recent matches and about specific matches from today and yesterday.
/matchmapstats/id This will give additional information about a specific map in a match.

uncategorized data

/streams This gives a list of current csgo stream.
